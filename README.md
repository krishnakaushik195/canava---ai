## 🛠️ Global Application Utilities
* `main.py` & `app.py`: Core routing, entry points, and orchestration layers for running the automated generations.
* `Booklet_merger.py`: Document consolidation engine used to merge output PDF/image sequences.
* `Schulung_seven.py`: Training automation and specialized resource handler.
* `test.py`: Internal sandbox for validation runs.
